# Profit & Inventory Tracker

Track sales, inventory, and profit splits across multiple products when working with investor capital.

![Profit Tracker Screenshot](assets/profitTracker-view.png)

## Quick Start

Open `tracker.html` in your browser. No installation needed.

## Features

- **Multi-product tracking** - Manage unlimited products with different prices and capital structures
- **Split-view dashboard** - Compare two products side-by-side
- **Automatic profit splits** - Calculates your share vs. investor returns
- **Persistent storage** - Data saves to browser localStorage
- **Offline-first** - No server or internet connection required
